## 🚀 About Me
안녕하세요! 게임 개발의 세계에 발을 들인 **김인권**입니다.  
단순한 플레이를 넘어, 누군가에게 몰입감 넘치는 경험을 선사하는 게임을 직접 만들기 위해 공부하고 있습니다.  
현재는 **언리얼 엔진(Unreal Engine)**의 강력한 퍼포먼스에 매료되어 이를 깊이 있게 탐구 중입니다.

- 🌱 현재 **게임 개발 및 언리얼 엔진**을 집중적으로 학습하고 있습니다.
- 🎯 최종 목표는 타격감과 액션성이 살아있는 **Action RPG 게임**을 개발하는 것입니다.
- 💡 기술적인 구현뿐만 아니라, 사용자 경험(UX)을 고려한 설계를 고민합니다.
