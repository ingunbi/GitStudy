## 🗺️ Action RPG Development Roadmap

### Phase 1: Foundation (Current)
- [x] Unreal Engine 5 환경 설정 및 Git 연동
- [ ] C++ 기초 및 언리얼 매크로(UProperty, UFunction) 숙달
- [ ] Enhanced Input System을 활용한 캐릭터 이동 구현

### Phase 2: Core Combat System
- [ ] 가로/세로 공격 애니메이션 몽타주(Montage) 제작
- [ ] 충돌 판정(Line Trace / Collision) 및 데미지 처리 시스템
- [ ] 상태 머신(FSM) 기반의 에너미 AI 기초 설계

### Phase 3: Polish & Narrative
- [ ] 나이아가라(Niagara)를 활용한 타격 이펙트 최적화
- [ ] 게임의 몰입감을 더하는 시네마틱 및 스토리 연출 (Sequencer)
- [ ] 최종 빌드 및 포트폴리오 정리
